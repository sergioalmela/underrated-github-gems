### ü•á #1 - pasteguard
**Repo:** [`sgasser/pasteguard`](https://github.com/sgasser/pasteguard)
**Language:** TypeScript
**Why it stands out:** A privacy proxy that sanitizes PII before it reaches any LLM, directly addressing the "shadow AI" compliance crisis in companies. **Highlights:**

- Presidio-based entity detection
- Addresses real enterprise compliance needs
- Clean TypeScript architecture, ready to deploy

### ü•à #2 - coi
**Repo:** [`io-eric/coi`](https://github.com/io-eric/coi)
**Language:** C++
**Why it stands out:** A bold new programming language purpose-built for high-performance web apps with WASM and fine-grained reactivity. Instead of adding yet another framework on top of JavaScript, Coi tackles frontend performance at the compiler level with type safety and deterministic performance guarantees. A technically stunning achievement for compiler and web architecture enthusiasts.

**Highlights:**
- Custom compiler targeting WebAssembly
- Fine-grained reactivity built into the language
- Type-safe with deterministic performance
- MIT licensed with active development

### ü•â #3 - unsloth-mlx
**Repo:** [`ARahim3/unsloth-mlx`](https://github.com/ARahim3/unsloth-mlx)
**Language:** Python
**Why it stands out:** Brings the popular Unsloth LLM training optimizations to Apple Silicon via the MLX framework. Before this, Mac users were largely excluded from efficient model fine-tuning. This project opens the door for a massive user base to train and customize models locally without needing NVIDIA hardware.

**Highlights:**
- Unsloth optimizations ported to Apple's MLX
- Enables LLM fine-tuning on Mac hardware
- Removes NVIDIA dependency for training
- Python-based, accessible to ML practitioners
