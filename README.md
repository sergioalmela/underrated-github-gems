## 📊 How We Pick Gems

We look for:

- ⭐ Less than **500 stars**
- 💡 Original ideas or clever execution
- 📖 Great documentation or clean UX
- 🔄 Actively maintained and usable

No low-effort projects. No abandoned ideas. Just pure dev joy.
